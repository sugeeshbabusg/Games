Instructions
1.Install Pydroid 3 on your smartphone.
2.Make sure pygame is installed on it.
3.Open the file pingpong.py in the app.
4.Run the game in landscape mode. 
5.Make sure a external keyboard is connected to the smartphone for playing the game. Use arrow keys to control the player.
6.If there is a lag in gameplay, it may be due to your screen resolution. Tweak the game code to fix the problem.
